# password-security-toolkit
A Python + Flask web app to generate and analyse password strength.

---

## 🚀 Features  

- Password Generator (8–64 length)  
- Strength Checker (score 0–10)  
- Weak / Medium / Strong indicator  
- Suggestions for better passwords  
- Stores last 3 passwords  

---

## 🛠️ Tech Stack  

- Python  
- Flask  
- HTML / CSS / JavaScript  

---

## ▶️ How to Run  

1. Install Flask  
pip install flask  

2. Run app  
python flask_app.py  

3. Open browser  
http://127.0.0.1:5000/
